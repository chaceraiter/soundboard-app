# soundboard-app

Simple iOS soundboard prototype (SwiftUI).

## Run
- Open `SoundboardApp.xcodeproj` in Xcode.
- Select a simulator (iPhone) and press Run.
- If prompted, accept the Xcode license by launching Xcode (or run `sudo xcodebuild -license` in Terminal).

## Add sounds
- Drop `.wav` files into `SoundboardApp/Resources/Sounds/`.
- Update the `items` list in `SoundboardApp/ContentView.swift`.
